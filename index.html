Uma biblioteca simples e poderosa para intergrasse em aplicações C# de diversos escopos: Web, Mobile, Desktop. A CisonExtractionEmail está inserida dentro do Cison.Text que é uma biblioteca completa para manipulação de strings. Para instalar a biblioteca via CLI utilize a seguinte linha de código:
dotnet add package CisonExtractionEmail
